# Welcome to your project

**About**

This project contains everything you need to run your app locally.

**Edit the code in your local development environment**

Any change pushed to the repo will be reflected in your production environment.

**Prerequisites:** 

1. Clone the repository using the project's Git URL 
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file if needed for your environment variables

Run the app: `npm run dev`

**Publish your changes**

Push your changes to GitHub to trigger a new deployment.
